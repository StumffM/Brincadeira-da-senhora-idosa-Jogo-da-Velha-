Este projeto implementa um Jogo da Velha simples em Java, foi meu primeiro desafio mais "complexo" de programação fora da Faculdade.
E contem muito do que aprendi de Fundamentos de Programação.

Estrutura dos Arquivos:

* App.java - Responsável pelo menu principal, controle das rodadas e gerenciamento da vez dos jogadores.

* Player.java - Contém a lógica relacionada aos jogadores, como identificação e controle das jogadas.

* Verifica.java - Realiza as validações do jogo, verificando:
  - Se uma posição já está ocupada
  - Se houve vitória em alguma rodada
  - Se o jogo terminou em empate

* Imprime.java - Responsável pela impressão e atualização visual do tabuleiro no console.
